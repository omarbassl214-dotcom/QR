# Perfect Protocol System

A premium, luxury QR-based guest management and timeline system designed for high-end events.

## Features

- **Premium UI/UX**: Dark mode, glassmorphism, and smooth micro-animations.
- **Dynamic QR Scanner**: Advanced guest identification and check-in.
- **Interactive Timelines**: Radial orbital timelines for event scheduling.
- **Event Management**: Specialized layouts for weddings and corporate events.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com) 4.0
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **QR Code**: [qrcode.react](https://github.com/zpao/qrcode.react)

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/perfect-protocol-system.git
   cd perfect-protocol-system
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   - Copy `.env.example` to `.env.local`
   - Fill in the required values.

4. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

### Vercel (Recommended)

The easiest way to deploy is through the [Vercel Platform](https://vercel.com/new).

1. Push your code to GitHub.
2. Import the project into Vercel.
3. Add your environment variables in the Vercel dashboard.
4. Deploy!
